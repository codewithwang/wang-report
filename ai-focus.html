By Aya NakamuraAI Desk, Senior CorrespondentTue, Jun 2

Anthropic's S-1 filing this week names what the lab is: a deployment business with a $47 billion annualized revenue run rate, converting enterprise API contracts into the next training run on B200 NVL72 clusters that BIS October 2023 Export Administration Regulations keep out of PRC data centers. Opus 4.8, which independent reviewers benchmark as a measurable improvement on GPQA-Diamond and SWE-bench relative to Claude 3 Opus, ships into an install base the prospectus must now legally describe. The increment proves the pipeline is live. The pipeline pays for the cluster. Anthropic's S-1 makes both facts a disclosure obligation to the SEC.

On the PRC side, Baidu, Zhipu AI, and DeepSeek are each training on Ascend 910B clusters. None of the three has a Bedrock equivalent. That absence is the structural fact: no commercial API deployment surface means no enterprise billings compounding into cluster-expansion capital the way Anthropic's Bedrock footprint does through AWS Singapore ap-southeast-1, which means no self-funding mechanism for cluster expansion against the BIS October 2023 hardware differential. Anthropic's $47 billion annualized run rate measures how far that differential has compounded since October 2023. Cognition's $1 billion raise at $26 billion valuation this week, for agentic deployment on B200 NVL72 infrastructure, adds a second commercial loop to the same side of the bilateral that Anthropic's S-1 now prices.

The second paragraph does the structural work the first paragraph sets up. Nakamura has found the number that converts a hardware differential into a capital mechanic. — WR
